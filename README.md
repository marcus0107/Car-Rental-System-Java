 Car Rental System (Java OOP)

 Overview
This project is a Car Rental System developed using Object-Oriented Programming principles in Java. It allows a rental agency to manage cars, customers, and rental transactions.

 Features
- Add cars and customers
- Rent a car
- Return a car
- View available cars
- Menu-driven interface

 OOP Concepts Used
- Encapsulation
- Abstraction
- Modularity
- Association

 How to Run
1. Compile all Java files.
2. Run Main.java
